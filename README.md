## 📌설치 및 실행 방법
로컬 환경에서 이 프로젝트를 실행하려면 다음 단계를 따르세요.

### 사전 요구사항
- Node.js (v14 이상)
- npm (v6 이상)

### 1️⃣설치
1. 레포지토리를 클론합니다.
    ```sh
    git clone https://github.com/username/repository-name.git
    cd repository-name
    ```

2. 필요한 패키지를 설치합니다.
    ```sh
    npm install
    ```

### 2️⃣실행
개발 서버를 실행합니다.
```sh
npm start
```
브라우저에서 http://localhost:3000을 열어 프로젝트를 확인하세요.
